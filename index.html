Work and Projects

Framework for Efficient Development of Secure Multiparty Computation

At MIT Lincoln Lab, I am designing and developing a framework for secure multiparty computation (MPC) protocols. MPC is a privacy-enhancing, cryptographic technique for securely computing functions on private data - that is, the inputs and all intermediate values are never revealed to anyone. Our framework is focused on improving the speed at which developers can implement MPC protocols and applications.

Externally Verifiable Oblivious Random Access Memory

Oblivious Random Access Memory (ORAM) protocols are cryptographic protocols for obscuring a client’s access patterns to a set of data. In this project, we extended existing ORAM protocols to suppport a public, external party (e.g., a friend, paid contractor, or smart contract on a blockchain) to verify that the client and server are adhering to the protocol. If a party deviates from the protocol, then the victim party can appeal to the third party who can determine with certainty, using the underlying cryptography, that either the accused party deviated from the protocol or that the accuser lied.

This is joint work with Adam Groce and Josh Gancher. Our paper can be found on ePrint.

Component-based Garbled Circuits

Yao’s garbled circuits is a cryptographic technique for enabling two parties to jointly and securely compute a function while keeping their inputs private. In this project, we improved the performance and flexibility of garbled circuits by creating many circuits for common tasks (e.g., addition, equality, less-than) in an offline phase and linking the circuits together in the online phase to quickly create a larger circuit for a specific functionality.

We implemented our work in two libraries. I was the primary author of CompGC, which links common sub-circuits into larger circuits as described above. The second library, libgarble, was primarily written by Alex Malozemoff and is for executing garbled circuits.

This was joint work with Adam Groce, Arkady Yerukhimovich, and Alex Malozemoff. Our paper can be found on ePrint.

Identifying Structural Variants in Genomic Data using Community Detection

I worked with Dr. Anna Ritz on finding and categorizing structural variants in genomic data using community detection graph algorithms. My python library implements Blondel et al.’s community detection algorithm.

Evolution in the Patents Database

I worked with Dr. Mark Bedau and others on investigating the U.S. patents database to analyze the evolution of technology over time.
